GOA/Mouse Load: Inputs, Logs, etc.

    Inputs

    • GOA File (sorted)
    • Uberon File
    • Annotations to Load into MGI
    • Annotations to Append to MGI Gene Association file

    Reports

    Annotations in these files are neither loaded into MGI nor appended to the MGI Gene Association file.
    • GOA Annotations that originated from MGI
    • GOA Annotations that are duplicates to those in MGI
    • GOA Annotations of PubMed IDs that are not in MGI
    • PubMed IDs from GOA that are not in MGI
    • GOA Annotations not loaded because there is no PubMed ID
    • GOA Annotations where UniProtID resolves to more than one MGI Marker
    • GOA Annotations where UniProtID does not resolve to a MGI Marker
    • GOA Annotations where GO id is a root id (GO:0003674, GO:0008150, GO:0005575)
    • Uberon/EMAPA translations
    • Uberon ID errors
