    Reports

    • This file reports action taken by load for each line of the input.
      • Reason Codes:
      1. ANNOT_IN_MGI_TO - rows that contain an existing marker/go term/ISO/inferred-from annotation for J:73065 are skipped
      2. NON_UPKB - column 1 != "UniProtKB" (there are currently none of these)
      3. EXCL_GO - column 5 contains an excluded GO id (GO:0005515 or GO:0005488)
      4. EXCL_EVID - column 7 contains an excluded Evidence Code MGI - column 15 contains an assigned by value = 'MGI'
      5. NO_PMID - column 6 contains no PubMed ID
      6. NO_HOM - no human/mouse homology by HomoloGene class
      7. IS_GOA_NOT - column 4 contains a NOT qualifier . this will prevent transfer of the same GO term from other members of the same HomoloGene Class
      8. IS_MGI_NOT - mouse has a "NOT" GO annotation in MGI
      9. NON_1TO1_P - homology is not 1:1 and GO ID is Bio Process
      10. NOT_NO_TRANSFER - results from IS_GOA_NOT above
      11. DUP_IN_INPUT - this line caused is a duplicate annotation and was filtered out
      12. CREATE_ANNOT - annotation created
      13. CREATE_ANNOT_COL - this line's properties collapsed with another (for collapsed annotations, the first line is labeled CREATE_ANNOT, subsequent lines are labeled CREATE_ANNOT_COL)
